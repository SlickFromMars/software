TSA Software Development

What Is This Competition About?

Using leadership and 21st century skills, participants apply knowledge of cutting-edge technologies and algorithms to design, implement, test, and document a software development project.

What Will You Create?

Prior to competition, participants identify a societal need and design a tool (e.g., graphical user interface, dashboard, predictive model, etc.) using software of choice that addresses this need. On site, you will present this software to a panel of judges.

Skills You Will Use

This event provides opportunity for students to build and develop leadership and 21st century skills such as: Communication, Collaboration/Social Skills, Initiative, Problem Solving/Risk Taking, Critical Thinking, Perseverance/Grit, Creativity, Relationship Building/Teamwork, Dependability/Integrity, and Flexibility/Adaptability.

Career Connections and Other Competitions

This event has connections to various careers, including graphic designer and software engineer. If you are interested in this, here are some other TSA competitions you might be interested in: Software Development, System Control Technology, Video Game Design, Virtual Reality Visualization, Webmaster.
